# Taskany Bricks

This is a package with components, which we use in our projects.

## Install

```shell
$ npm i @taskany/bricks
```

## Usage

```jsx
import { Text } from '@taskany/bricks';

const App = () => <Text>Taskany Text Component</Text>;
```
